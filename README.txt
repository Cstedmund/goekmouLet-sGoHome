If your computer do not have nodejs before, please install it first by this command

Sudo npm install -g express-generator

After you got the nodejs, just enter the game folder and run the following command in the terminal

cd <../nodejs/express/goekmou>
DEBUG=goekmou:* npm start

Open a broswer and enter this URL: localhost:3000/maingame.html

End the Server by pressing control C in terminal
